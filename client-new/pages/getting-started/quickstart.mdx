# Quick Start

Get up and running with Dojo in minutes.

## Create a New Project

```bash
dojo new my-first-game
cd my-first-game
```

## Project Structure

Your new Dojo project contains:

- `src/` - Cairo smart contracts
- `models/` - Game state models
- `systems/` - Game logic systems
- `scripts/` - Deployment scripts

## Build and Run

```bash
# Build the project
dojo build

# Start local development node
katana --dev

# Deploy to local network
dojo deploy
```
